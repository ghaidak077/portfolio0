The Functional Layer (Interaction Map)How the code talks to itself.A. The Font Chainindex.html → links style.css → @font-face imports ../fonts/Syne.woff2.Risk: If the font folder is named Font instead of fonts (case sensitive), typography breaks.B. The Scroll ChainUser Scrolls → lenis.min.js smooths the value → script.js listens to the value.If scrollY > 50px: Nav becomes solid black.If scrollY > 500px: "Back to Top" button appears.C. The Image Swap Chain (The Complex Part)User hovers or scrolls to .project-item (e.g., Novatech).HTML Element has data-bg="bg-1".script.js reads this ID.script.js finds <img id="bg-1"> in the .curtain-layer.Action: Adds class .active-project to that image (Opacity: 0.3).Action: Removes .active-project from all other images.
